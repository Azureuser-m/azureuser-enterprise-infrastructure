**Limitation:** The official `zabbix.zabbix.agent` Ansible role was not usable on the Proxmox host because the host runs on Debian 13 (Trixie), while the role’s documented support matrix currently lists Debian 10/11/12 and the collection is tested with Ansible Core 2.17/2.18. As a result, Zabbix agent deployment on Proxmox was performed using the official Zabbix Debian 13 repository and a custom/manual Ansible playbook instead of the official role.
